CSS Question and Answers


What are main benefits of using CSS?

-CSS offers better control of formatting than the alternative provided in HTML
-Websites are more maintainable since all formatting can be centralized into one or few CSS files
-Websites that implement CSS are more accessible
-Sites that use a centralized set of CSS files for presentations will be quicker since each HTML file will contain less markup
-CSS can be used to adopt a page for different output mediums



What are different parts of a CSS style rule?

Selector: identifies which element or elements in the HTML document will be affected by the declaration rule
Property: part of a CSS declaration inwhich a property name is predefined by the CSS standard ie. property type "Fonts" properties are "font", "font-family" etc...
Value: part of a CSS declaration which is dependent on the property. These can be from a predefined keyword list or others include length measurements, percentages, color values, URLs etc...



What is a class selector? Give an example

A class selector allows for the simultaneous targetting of different HTML elements regardless of the position within a document by using the period (.) and the class name
Example:
    <style>
       .example {
         width: 500px;
         height: 100px;
         border: 1px solid #000;
       }
    </style>

What is an id selector? Give an example

An id selector allows for the selection of a specific element by its id attribute regardless of type or position and can be targeted for styling by using the id selector with pound/hash (#) and id name
Example:
    <style>
       #example {
         width: 500px;
         height: 100px;
         border: 1px solid #000;
         font-size: 16pt;
       }
    </style>

What is an attribute selector? Give an example

An attrubute selector allows you to select HTML elements by either presence of an element attribute or by value of an attribute
Example:
       [title] {
       padding-bottom: 5px;
       border-bottom: 3px dashed red;
       text-decoration: underline;
     }



What is a pseudo selector? Give an example

A pseudo selector is a way to select something that doesn't exist explicitly as an element but still a recognizable selectable object
Example: a:hover {
     text-decoration: underline;
     color: violet;
     font-weight: bold;
     }

What are the three cascade principles used by browsers when style rules conflict?

Inheritance: CSS properties can affect their descendants as well
Specificity: is how the browser determines which style rule takes priority when more than one style rule could be applied to the same element
Location: is used when inheritance and specificity cannot determine style precedence - latest given more weight
